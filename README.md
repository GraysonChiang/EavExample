#  Magento 2 EAV Entity Example


## 執行環境
適用版本：Magento 2.0.0 以上


## 說明
* 新增 Magento 2 EAV Entity Model 的範例程式


## 安裝

1. 下載原始檔，放置於 `app/code/Grayson/Post`(若無此資料夾請自行新增)資料夾內
2. 執行 `bin/magento setup:upgrade` 指令
